# Church PWA

Senior 친화형 교회 소식 웹앱 예시입니다.

## 포함된 화면

- 홈
- 공지사항
- 기도제목
- 성경공부
- 교회소식
- 관리자 페이지 예시
- 관리자 입력 폼 미리보기

## 실행 방법

1. Node.js와 npm 또는 pnpm을 준비합니다.
2. 아래 명령으로 의존성을 설치합니다.

```bash
npm install
```

3. 개발 서버를 실행합니다.

```bash
npm run dev
```

4. 브라우저에서 `http://localhost:3000`을 엽니다.

## 폰에서 접속하는 방법

1. 노트북과 폰이 같은 Wi-Fi에 연결되어 있어야 합니다.
2. 개발 서버를 실행합니다.

```bash
npm run dev
```

3. 폰 브라우저에서 노트북의 로컬 주소로 접속합니다.

```text
http://192.168.1.151:3000
```

참고:

- `dev` 스크립트는 기본적으로 `0.0.0.0`으로 열리도록 설정되어 있습니다.
- 로컬 IP 주소는 네트워크가 바뀌면 달라질 수 있습니다.

## 현재 이 워크스페이스 상태

- 앱 코드는 생성되어 있습니다.
- 현재 Codex 실행 환경에는 `npm`이 포함되어 있지 않아 여기서 직접 설치 실행까지는 확인하지 못했습니다.
- 로컬에 npm이 준비되면 바로 `npm install` 후 실행 가능합니다.
